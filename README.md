# 🌍 Universal Converter Project

A modern, portfolio-ready Currency & Unit Converter built with vanilla HTML, CSS, and JavaScript.
It supports real-time currency exchange rates, multiple unit categories, automatic updates, and a modular project architecture designed for scalability.

The project demonstrates API integration, modular JavaScript structure, and responsive UI design, making it a solid portfolio project for web development.


## 📝 What Does It Do?

Universal Converter provides a clean interface that allows users to convert between currencies and measurement units instantly.

Main features include:

• Convert currencies using real-time exchange rates from an external API
• Convert multiple unit categories such as length, weight, and more
• Instantly swap source and target units
• Automatic currency rate updates every 10 minutes
• Clean and responsive UI for fast conversions
• Modular structure allowing easy addition of new units or categories


## ⚙️ How Does It Work?

The application is built using core web technologies and structured modules.

HTML Structure – Semantic layout containing the converter interface, input fields, dropdown selectors, and result display.

CSS Styling – Responsive and modern UI design with clear layout and user-friendly interaction.

Vanilla JavaScript – Handles DOM manipulation, event listeners, dynamic category loading, conversion logic, and swap functionality.

Conversion Engine – A reusable conversion utility calculates values based on unit rates.

Currency API Integration – Fetches live exchange rates from an exchange rate API to ensure accurate currency conversions.

Automatic Updates – Currency data refreshes every 10 minutes to keep exchange rates current.

Modular Architecture – The project separates logic into modules such as:

api/
data/
utils/
This makes the application easier to maintain and extend.

Deployment – The project can be deployed easily using platforms like Vercel, Netlify, or GitHub Pages.
